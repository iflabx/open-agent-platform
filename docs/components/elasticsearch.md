# Elasticsearch

> 类型：全文检索与混合检索兼容项
>
> 复用规则：稳定复用的组件说明页

## 当前定位

`Elasticsearch` 在当前方案中作为全文检索与混合检索兼容项，用于特定主平台路线或全文搜索要求较高的场景。

## 主要职责

- 全文检索
- 关键词检索
- 混合检索兼容支撑

## 适合场景

- 全文搜索要求较高的知识场景
- 需要与特定平台路线配套的检索场景
- 全文与向量混合检索需求

## 边界

- 不作为当前默认主检索底座
- 存在许可证边界，正式落地前应结合 [开源状态与许可证说明](/open-source-status) 单独确认
- 若要求更低法律风险，可优先评估 `OpenSearch`
