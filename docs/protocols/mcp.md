# MCP 协议

> 类型：agent-to-tool / context 协议
>
> 复用规则：稳定复用的协议说明页
>
> 官方网站：https://modelcontextprotocol.io/
>
> 开源仓库地址：https://github.com/modelcontextprotocol/modelcontextprotocol

## 当前定位

`MCP` 在当前方案中作为工具、资源、提示词和上下文能力的标准接入协议。

## 主要职责

- 标准化工具接入
- 标准化上下文与资源接入
- 降低主平台与外部能力的集成耦合

## 适用场景

- 外部系统工具接入
- 检索资源暴露
- Prompt / context 能力标准化

## 边界

- 不负责 agent-to-agent 协作
- 不替代权限治理和审计
